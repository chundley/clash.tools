New Features

A summary of the theme for this release

New Start War Interface

The interface for starting a new war has been revamped. It's now easier and more efficient to get the next war started. You can now use the previous war roster as a starting point and can also use your team's saved weights to automatically generate rosters.

Smaller updates and bug fixes

  • The end war function for leaders has been made more mobile friendly and bugs have been fixed that sometimes prevented clans from ending a war.
  • Prevously clan leaders couldn't leave the clan even if they were the last person. This caused problem when multiple people created the same clan and one wanted to leave to join the other. You can now leave the clan even if you are the last person in it.
  • Image upload was broken after adding a load balancer to the system. The file size limit was incorrectly set and has been fixed.
  • War weight in the member profile, arranged wars, and war roster were limited to a high value of 99. This has been changed to 100 for maxed TH10's.
  • The giant and goblin were out of order in the profile view - this has been fixed.
